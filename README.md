## 认识自我
> 如果我的人生是一幅画或一支曲，我希望它是在表达一个主题。

- 成为美，感受美，创造美。
- 成为爱，感受爱，创造爱。

隐含假设：1. 源头是美，源头是爱。 2. 美和爱才是真实的。
