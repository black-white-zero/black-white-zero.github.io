## 认识自我
> 如果我的人生是一幅画或一支曲，我希望它是在表达一个主题。

- 成为美，感受美。
- 成为爱，感受爱。

隐含假设：1. 源头是美，源头是爱。 2. 美和爱才是真实的。

## 认识自我和世界的关系
> 在我落幕前，希望可以自豪的说一声：“我给世界留下了两份礼物：一份是我的人生，一份是我的作品。”
